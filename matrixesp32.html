Matrix-ESP32

Matrix-ESP32 is a very basic Matrix client for the TTGO T-display microcontroller. I have really grown to love developing for these little boards. With the complexity of modern computers, development on this board is a breath of fresh air. Knowing that when you power it on, it only is doing exactly what you told it to, and nothing else.

This project is interfacing with the Matrix Client-Server API, containing the ability to authenticate a user's account and connect to homeservers. It mostly allows for basic read-only features, such as viewing messages in the user's saved rooms.
